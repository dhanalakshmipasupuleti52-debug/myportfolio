Tourist Places Website

This project is a Tourist Places website
developed using HTML and CSS.

The main purpose of this website is to
display famous tourist destinations
in a simple and attractive layout.

The website includes different sections
such as Home, Places, Tickets and Contact
to help users easily explore information.

Popular tourist locations like
Taj Mahal, Eiffel Tower,
Great Wall of China and Statue of Liberty
are presented using image cards
with descriptions.

Through this project, I practiced
website layout design, navigation
structure and image card styling.

This project helped me improve my
HTML and CSS skills while building
a simple travel information website.
